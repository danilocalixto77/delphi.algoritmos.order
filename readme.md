# Algoritmos de Ordenação em Delphi  
### Exercicios de lógica utilizando algoritmos de ordenação em Delphi/Pascal.
##### Versão do Delphi : Delphi 11 CE

- O Projeto visa desenvolver os principais algorítmos de ordenação na linguagem de programação Delphe/Object Pascal. Utilizando procedures, funções e procuraremos implementar classes e POO para melhoramento do código.

## Bubble Sort
- Primeiro algorítmo implementado com a utilização de laços for, procedures, funções e arrays.
- Este algorítimo consite em ordenar uma lista de valores(arrays), fazendo a comparação do valor atual com o valor seguinte, e caso o atual seja maior que o seguinte, a posição dos dois serão trocadas. Desta forma seguirá comparando até o final da lista e não havendo mais valores fora de ordenação crescente ou descrescente, o laço será finalizado.
